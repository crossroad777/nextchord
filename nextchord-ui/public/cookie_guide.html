YouTube ダウンロード設定ガイド

ロボット検出エラーを回避し、安全にYouTube音源を解析するためのクッキー設定手順

🔒 セキュリティとプライバシーについて

ローカル保存のみで安全：

入力されたクッキー情報は、サーバー側には一切保存されません。お客様のブラウザ内（LocalStorage）にのみ安全に保存され、解析リクエストを行う瞬間のみ一時的にYouTubeへのアクセス認証に使用されます。

サブアカウントでの利用を推奨：

安全のため、メインのGoogleアカウントではなく、テスト用（サブ）のGoogleアカウントでYouTubeにログインしてエクスポートすることを強くお勧めします。

🛠️ 事前準備：クッキー取得用ツールのインストール

ブラウザのクッキーを「Netscape形式」のテキストファイルとしてエクスポートするために、お使いのブラウザに合わせて以下の拡張機能（無料・オープンソース）をインストールします。

Chrome / Edge / Brave 等
「Get cookies.txt LOCALLY」を使用します。クッキーデータをローカル処理する安全なオープンソース拡張機能です。
Chromeウェブストアを開く
Firefox
「Export Cookies.txt」を使用します。1クリックでNetscape形式のクッキーを簡単にエクスポートできます。
Firefox Add-onsを開く

📝 クッキーのエクスポート手順

⚠️ 最重要：クッキーがすぐに切れてしまうのを防ぐには

クッキーを書き出した後に、同じブラウザでYouTubeを操作（クリックやリロードなど）すると、YouTubeのセキュリティによりセッションキーが自動更新され、エクスポートしたクッキーが即座に無効化されます。

この現象を防ぐため、以下の「シークレットウィンドウを使った手順」を正確に行ってください。

  • 1
    シークレット ウィンドウを新しく開く
    ブラウザのメニューから **シークレット ウィンドウ**（シークレットモード / プライベートブラウズ）を新しく開きます。
    ※インストールした拡張機能がシークレットモードでも動作するように、ブラウザの設定で拡張機能の「シークレットモードでの実行を許可」をあらかじめオンに設定しておいてください。
  • 2
    YouTubeにログインする
    シークレットウィンドウで YouTube にアクセスし、ログインします。（※サブアカウントでのログインを推奨）。
  • 3
    クッキーをエクスポートする
    ログイン後、YouTubeのトップページが表示された状態のまま（他のリンクをクリックしたり動画を再生したりせずに）、インストールした拡張機能をクリックします。
    表示されたテキスト領域の内容をすべて選択してコピー（Ctrl + A & Ctrl + C）します。
    ※テキストは # Netscape HTTP Cookie File で始まるフォーマットです。
  • 4
    シークレットウィンドウを「即座に」閉じる
    コピーが完了したら、すぐにそのシークレットウィンドウ（タブ）を完全に閉じます。他の操作を行わないことで、コピーしたクッキーの有効期限を保護します。

⚙️ NextChordへの設定手順

  • 1
    設定を開く
    NextChordの画面右上にある **設定アイコン（歯車マーク ⚙️）** をクリックします。
  • 2
    クッキーを貼り付ける
    「サーバー設定」モーダルが表示されます。「YouTube クッキー」のテキストエリアに、ステップ3でコピーしたテキストを貼り付けます。
  • 3
    保存して適用する
    右下の「保存して適用」ボタンをクリックします。これで設定が保存され、次回以降のYouTube解析でこのクッキーが使用されます。

❓ トラブルシューティング（Q&A）

Q. 「Sign in to confirm you're not a bot」エラーが表示される
登録したクッキーの期限が切れているか、エクスポート後にYouTubeを開いたため無効化されています。新しいシークレットウィンドウを開き、手順に沿って新しいクッキーをエクスポートし、NextChordに再度設定してください。
Q. シークレットウィンドウでクッキー拡張機能がクリックできない
ブラウザの設定により、シークレットモードでの拡張機能の利用が制限されています。ブラウザの設定ページ（例：Chromeの場合は chrome://extensions）を開き、対象の拡張機能の「詳細」をクリックし、「シークレットモードでの実行を許可」をオンにしてください。
Q. クッキーを設定したのにダウンロードできない動画がある
メンバー限定動画、年齢制限動画、または一部の非公開動画は、ログインしたアカウント自身がYouTube上でその動画を再生できる権限を持っていない場合、ダウンロードできません。対象の動画を再生できるアカウントでログインした上でクッキーをエクスポートしてください。
