# 🎸 NextChord — AI Chord Detection & TAB Generator

Upload any MP3/WAV file and get:

- 🎵 **Chord detection** with beat synchronization
- 📝 **Lyrics extraction** (Japanese & English)
- 🎸 **Guitar TAB** auto-generation
- 🎹 **Piano/Guitar chord diagrams**
- 📤 **Export** to MIDI, MusicXML, PDF, Text

## How to use

1. Click "Upload" and select an audio file
2. Wait for AI analysis (~30 seconds)
3. View chords, lyrics, and TAB in real-time playback
4. Export your results!
